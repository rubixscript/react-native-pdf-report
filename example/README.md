# RubixScript PDF Report - Example App

This is a minimal example demonstrating the `@rubixscript/react-native-pdf-report` library.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a:

- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go)

## What's included

This example demonstrates:

- Basic PDF report generation using `PDFReportModal` component
- Sample book and reading session data
- Report generation with customizable options
- Light/Dark theme support
